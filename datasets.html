Our Datasets


The datapackages listed here are full representations of the data you see on other parts of this website. The data is meant to be used however you see fit, hooray for open data. Some potential use cases we've thought of are:

  • Search Terms - when searching through large data dumps
  • A Glossary - of unusual terms that help provide context when researching
  • Machine Readible - list of terms and/or malware identifiers to import

If you're interested in automatically interacting with our datasets, you can also use our public API.

    More on Data Packages

    We hope that you will find these useful and that you will help contribute to expand upon this data. The data is stored in CSV with a JSON schema as based on data protocols spec created by the Open Knowledge Foundation. If you would like to learn more about creating data, OKFN has some easy to follow tutorials.
